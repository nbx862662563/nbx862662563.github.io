bind、call、apply区别及用法

发表于 2019-01-30 | 更新于 2019-02-16 | 分类于 JavaScript

开发过程中遇到的问题：

例1：

sayHi函数中的this指向 为 obj对象，但是obj中没有age属性，所以年龄为underfunded；

如果改为window.age，则输入ls18 岁了

例2：

db对象中没有sayHi方法，并且db对象不能直接访问obj对象中的方法；

但是调用的call、apply、bind方法：

作用：

​ call、apply、bind方法都可以改变函数的this指向，指向第一个参数；

区别：

​ 区别就是三者的参数、返回值不一样了：

​

可以看出，call和apply 的区别就是第二个参数不同：

​ call 的参数直接放进去….第二个，第三个…第n个…

​ apply 第二个参数为一个数组，参数存放在数组中

call 和 bind的区别为返回值的不同：

​ call的返回值是 函数的调用；

​ bind的返回值是 函数体，需要手动调用；
