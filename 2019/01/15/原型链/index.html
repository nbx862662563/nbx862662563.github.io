原型链

发表于 2019-01-15 | 更新于 2019-02-16 | 分类于 JavaScript

1.对象的声明方式

​ 第一种方式：字面量

​ 第二种方式：构造函数

​ 第三种方式：Object.create

2.构造函数

​ 构造函数 new ——-> 实例对象

​ 

构造函数通过   new   运算符   创建一个新对象，其中构造函数中的  this  指向  创建出来的实例

​ 实例对象 拥有 proto 属性

​ proto 指向 实例对象的 原型

​ 其中原型的 constructor 属性 指向 构造函数；

​ 圆形的 proto 属性 指向 Object构造函数的原型对象

​

​ 实例对象的原型 也 拥有 proto 属性

​ 实例对象的原型 的 原型对象，也是构造函数Object的原型对象，可以通过 Object.prototype 访问

​ Object的原型对象 有 constructor: ƒ Object() 属性 指向 Object构造函数

​ Object的原型对象的 proto 指向null

​ 打印结果为null

3.三角关系

​ 1.构造函数 new 实例对象；

​ 2.实例对象 proto 指向它的原型；

​ 3.实例对象的原型对象 可以通过constructor 查找 创建它的构造函数；

​ 4.实例对象的原型对象 的 proto 指向 Object 构造函数的 原型对象；

​ 5.Object 构造函数的原型对象 的 proto 指向null；
